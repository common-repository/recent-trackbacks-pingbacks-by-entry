Display recent trackbacks and/or pingbacks list simply on sidebar sort by entry.

== Description ==

Entry based recent trackbacks / pingbacks.

== Installation ==

Simply place the widget_recent_trackpingbacks_by_entry.php file in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!

If you want to display trackback and pingback separatelly, use widget_recent_trackbacks_by_entry.php and widget_recent_pingbacks_by_entry.php.

== Frequently Asked Questions ==

= What's difference between trackback and pingback? =

Pingback is default linkback between WordPress and another WordPress, and trackback is more common linkback such as MovableType and so on.
